# NRS 목표 페이지 — 핵심 인터랙션 플로우

## Flow 1: 뷰 전환 (리스트 ↔ 맵)
**트리거:** 상단 툴바 `맵` / `리스트` 탭 버튼

1. 초기 — 리스트 뷰 활성 (테이블)
2. `맵` 탭 클릭 → 카드 레이아웃 + 스피너 (`GET /goals/map/{orgId}`)
3. 맵 뷰 완성 — Objective 카드 / KR 카드 / Initiative 카드, 트리 선으로 연결
4. `리스트` 탭 → 테이블 복귀

**구현 힌트:** `aria-selected`로 활성 탭 식별, 전환 시 API 재호출

## Flow 2: 행 펼침/접힘 (Tree Expand/Collapse)
**트리거:** 행 좌측 `▼` / `▶` 토글 버튼

1. 초기 — Objective + 하위 모두 펼침 (depth 0/1/2)
2. Objective의 ▼ 클릭 → 하위 행 모두 숨김 (UI 로컬 상태)
3. 아이콘 ▼ → ▶ 변경
4. 재클릭 시 원복

**구현 힌트:** `goal.isOpen` 상태 관리, `parentId` 체인을 따라 visibility 결정

## Flow 3: 검색창 → 자동완성
**트리거:** 상단 검색 input 포커스

```
┌──────────────────────────────────┐
│ 내가 참여중인 목표                  │
│  · [아바타] 이름 / 팀명             │
├──────────────────────────────────┤
│ 전체 목표                          │
│  · [아바타] Objective 제목 / 팀명   │
└──────────────────────────────────┘
```

debounce 검색 → 항목 클릭 시 해당 목표로 스크롤/포커스 또는 새 탭 상세
